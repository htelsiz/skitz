# Claude

`claude` start interactive session ^run
`claude "{{prompt}}"` start with prompt ^run:prompt
`claude --help` show help ^run
`claude config list` list configuration ^run
`claude config set {{key}} {{value}}` set config ^run:key
`claude mcp list` list MCP servers ^run
`claude mcp serve` start MCP server ^run
`claude --model sonnet` use Sonnet model ^run
`claude --model opus` use Opus model ^run
`claude --print "{{prompt}}"` non-interactive mode ^run:prompt
`claude --continue` continue last conversation ^run
`claude --resume` resume a conversation ^run
