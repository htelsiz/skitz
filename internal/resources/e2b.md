# E2B

`e2b auth login` log in to E2B ^run
`e2b auth logout` log out of E2B ^run
`e2b sandbox list` list active sandboxes ^run
`e2b sandbox connect {{sandboxID}}` connect to a sandbox ^run:sandboxID
`e2b sandbox kill {{sandboxID}}` kill a sandbox ^run:sandboxID
`e2b template list` list templates ^run
`e2b template init` create a new template ^run
`e2b template build` build a template ^run
`e2b template delete {{templateID}}` delete a template ^run:templateID
